# AI 美甲试戴 Demo

React + Vite + Tailwind 的高保真交互 Demo，当前包含五个界面：

- 社区作品流
- 作品详情页
- AI 美甲试戴页
- 元素级细致编辑页
- 需求池占位页

## 运行

```bash
npm install
npm run dev
```

然后打开 Vite 输出的本地地址。当前开发环境的 PowerShell 能找到 Node.js，但找不到 `npm`、`pnpm` 或 `yarn`，所以本次没有在本机完成依赖安装和 dev server 启动。

## 素材

Demo 已从用户提供的 `hands.zip` 和 `styles.zip` 中抽取样本到：

- `public/assets/hands`
- `public/assets/styles`

试戴页可以通过右侧下拉菜单切换手部照片和款式照片。
